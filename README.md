# wally

Simple flutter application with firebase including

- Firebase Authentication(Google Signin)
- Cloud Firestore
- Image Uploading
- Firebase Storage
- Firebase ML Vision(Image Labeling)
- Firebase Analytics
- Firebase Dynamic Links
- Firebase Push Notifications
